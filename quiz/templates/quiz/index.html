Know the Rules!

1. The event will have 50 questions.
2. Solving a question is necessary in order to proceed to the next one.
3. +10 will be awarded for each correct answer.
4. No negative marking for wrong answers.
5. Answers to be in lowercase with no spaces and special characters. Numbers should be in digits.
6. Hints for a question will be provided in the club's facebook page if required.
